# Audiographica

Converts audio songs into sheet music notes.

![gif](./readme_media/demo.gif)

## Tech stack
* Back-end : Flask
* Front-end : React.js

## How to install

### For back-end
* Install dependencies in ./flask-api via `pip`

* Install lilypond >= 2.23.6 

### For front-end
* `npm install`
